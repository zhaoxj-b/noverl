# 各角色设计和进展（AI2AI）

## 1. 角色实时状态

### 韩莫莫（10岁/现在）
- **性格倾向**：敏感、忍受疼痛能力强、内心有深渊。
- **近期动作**：剪断长发反抗，发现爷爷书籍，开始进入连续回忆段。
- **状态**：第二部“十年旧事”进行中。

### 母亲
- **性格倾向**：面子大于天，控制欲与贫困交织的焦躁。将“体面”视为生存唯一准则。
- **近期动作**：在灵堂通过逼哭、按头等方式规训莫莫，在后院继续通过繁重家务维持“能干”形象。
- **状态**：活跃。

### 父亲
- **性格倾向**：二婚背景下的卑微与沉默。明面上懦弱避事，暗地里通过小物件（发卡、的确良、半块红薯）表达对莫莫的微弱温情。
- **近期动作**：在屋檐下吸旱烟回避冲突，在柴房偷偷给莫莫送东西。
- **状态**：活跃。

### 姑姑（韩家小姑）
- **性格倾向**：善良、沉默的庇护者。手部粗糙（裂口），曾从牛圈救出莫莫，是莫莫童年唯一的微弱亮光。
- **近期动作**：在灵堂给莫莫递热水，阻拦母亲施压。
- **状态**：活跃。

### 大姑（大舅妈/远房亲戚性质的大姑）
- **性格倾向**：刻薄、虚伪、爱嚼舌根。社会评价的具象化，与母亲的“体面”形成呼应与监督。
- **近期动作**：在灵堂议论莫莫“心狠”，在后院讥讽莫莫干活像“没骨头的蛇”。
- **状态**：活跃。

### 姐姐
- **性格倾向**：已碎裂的镜子。莫莫的恐惧源头。
- **背景事实**：手掌已被冲床压烂，留下终身残疾。
- **作用**：预示莫莫如果逃不出去的命运。

## 2. 性格一致性检查
- **母亲**：对话必须带“丢人/规矩/别人说/体面”。
- **父亲**：动作必须有“低头/避开眼神/偷偷摸摸的温情”。
- **姑姑**：动作重于语言，表现为“粗糙的手、沉默的扶住”。
- **大姑/亲戚**：语气阴阳怪气，专注于对他人的审判。
